## 准备篇

- 投资策略总的来说分为两大类：**投资趋势**和**投资波动**
- 依靠预测方向带来利润的策略，称之为**Alpha策略**，依靠波动率带来利润的策略，称之为**Beta策略**
- 量化投资是**主动型投资**，理论基础在于**弱有效市场假说**
- 策略收益可以是**绝对收益**——总体是赚钱的，或者是**相对收益**——高于市场平均收益
- 策略由**投资策略**——理论收益，**交易策略**——冲击成本，**风控策略**——风险阻断，三者组合而成
- 资产配置一般包括**两大类别**——战略资产配置和战术资产配置，**三大层次**——全球资产配置、大类资产配置和行业风格配置

## 策略篇

- 量化选股

  总的可以分为三大类**公司估值法**——价值决定价格，**趋势法**——运动具有惯性，**资金流**——主力推动行情

  - 多因子模型

    - 相对比较稳定，总有一些因子会发挥作用
    - 判断方法：打分法——比较稳健不易受极端值影响，回归法——能及时调整因子的敏感度但是容易受到极端值影响
    - 建模过程：
      1. 候选因子的选取：主要依赖经济逻辑和市场经验
      2. 因子有效性的检验：因子排序，因子分组，高分组应该持续跑赢比较基准、低分组应该持续跑输比较基准，这样才能说明因子具有一定的有效性
      3. 冗余因子的剔除：根据相关性等
      4. 综合评分选股：构建投资组合进行投资
      5. 下一投资周期因子的更新：因子的从新筛选——全部或部分

  - 风格轮动模型

    - 市场偏好

    - 风格鉴别

      1. 持股基础判别法——晨星风格箱法

      |      | 价值型   | 混合型   | 成长型   |
      | ---- | -------- | -------- | -------- |
      | 大盘 | 大盘价值 | 大盘混合 | 大盘成长 |
      | 中盘 | 中盘价值 | 中盘混合 | 中盘成长 |
      | 小盘 | 小盘价值 | 小盘混合 | 小盘成长 |

      规模指标：市值

      估值指标：平均市盈率、平均市净率

      2. 夏普收益率基础的投资风格鉴别——标普500成份股法

      | 股票            | 风格                                                         |
      | --------------- | ------------------------------------------------------------ |
      | 标普500成份股   | 按净市比（B/P）排序分两类（分界点是两类股票总市值相等）：价值股、成长股 |
      | 非标普500成份股 | 按市值高低分两类：中市值股（前80%）、小市值股（后20%）       |

    - 风格策略：按风格分组进行组合收益率计算

      1. 风格动量：收益率高的风格组-收益率低的风格组构成一个套利组合，观测持续性与检验统计显著性
      2. 风格反转：收益率低的风格组-收益率高的风格组构成一个套利组合，观测持续性与检验统计显著性

  - 行业轮动模型

    - 经济周期
    - 行业分为周期行业和非周期行业：用CAPM计算行业组合对比较基准的Beta值分类，大于1的为周期行业，小于1的为非周期行业
    - 行业轮动策略：货币政策M2扩张时配置周期行业，紧缩时配置非周期行业

  - 资金流模型

    - 资金推动
    - 资金流指标：招商资金流指标（CMSMF）
    - **股票池初选：剔除上市不满一个月的股票，剔除ST股票，剔除调仓期涨跌停及停牌的股票，剔除信息含量小于10%的股票**

- 量化择时
  - 趋势择时
    - 思想：技术分析
      1. 假设前提——市场行为包含一切信息、价格沿趋势运动、历史会重演
      2. 五类方法——指标类、切线类、形态类、K线类、波浪类
    - 指标：MA、MACD、DMA
  - 市场情绪择时
    - 思想：投资者的热情程度
    - 指标1：封闭式基金折价率、转股溢价率、IPO首日涨跌幅、IPO发现PE、上涨家数百分比、混合型基金平均仓位、股票型基金平均仓位
    - 指标2：市场情绪指数、情绪变化指数——用指标1的7个指标回归构建
  - 时变夏普率模型
  - 牛熊线模型
  - Hurst指数模型
    - 思想：分形理论在趋势拐点判断中的应用
  - SVM模型
  - SWARCH模型
    - 思想：利用宏观经济指标判断大盘
  - 异常指标择时
    - 思想：某些特殊情况下一些指标容易出现异常
    - 指标：市场噪声、行业集中度、兴登堡凶兆
